<div>
  <h1>🌤️ Weather App</h1>

  <p>
    <strong>React Native | Expo | TypeScript | TanStack Query | OpenWeather API</strong>
  </p>

  <p>
    A modern mobile weather application built with a scalable architecture using React Native (Expo), TanStack Query for caching, and Axios for API management.
  </p>

  <p>
    This project demonstrates mobile development, API integration, state management, and clean architecture design.
  </p>

  <p>
    <strong>Tech Stack:</strong><br>
    React Native • Expo • TypeScript • Axios • TanStack Query • OpenWeather API
  </p>

  <p>
    <strong>Features:</strong><br>
    • Real-time weather data<br>
    • City-based search<br>
    • API caching with TanStack Query<br>
    • Clean reusable component structure<br>
    • Mobile-first responsive UI
  </p>

  <p>
    <strong>API Used:</strong> OpenWeather API
  </p>

  <p>
    <strong>Architecture:</strong><br>
    - Modular API layer (Axios)<br>
    - Custom hooks for data fetching<br>
    - Component-based UI<br>
    - Query caching with TanStack Query
  </p>

  <p>
    <strong>Project Purpose:</strong><br>
    Built to demonstrate mobile application development skills, API integration, and production-style React Native architecture.
  </p>

  <p>
    <strong>Future Improvements:</strong><br>
    • Location-based weather (GPS)<br>
    • 7-day forecast view<br>
    • Weather animations<br>
    • Dark mode support<br>
    • Offline caching
  </p>
